In this project, I implemented Object-Oriented Programming (OOP) principles in Python to model a book entity. 
I utilized custom classes, instantiation, and attribute encapsulation. The program features structured class methods to handle internal data 
and employs modern string interpolation (f-strings) to dynamically print descriptive text directly from the object instance.

# Book Management System (Python OOP)

A lightweight, Object-Oriented Python application designed to model and manage book structures using classes, methods, and instances.

## Features
* **Object-Oriented Design:** Uses classes to blue-print real-world items like books, authors, or libraries.
* **Encapsulation:** Keeps properties and attributes safely structured within specific instances.
* **Dynamic Formatting:** Leverages modern string formatting to output clean text statuses to the console.

## Technical Concepts Used
* **Class Attributes (`self.title`):** References specific instance properties belonging to the object scope via the `self` keyword.
* **String Interpolation (F-strings):** Evaluates variables dynamically inside string literals `{}` at runtime for optimal text rendering.
* **Dunder Methods (`__str__` / `__repr__`):** Integrates standard string representation returns for user-friendly object printouts.
